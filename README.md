
  ## StudyBuddy (React Native + Expo)

  This project now runs as a React Native app using Expo.

  ## Run the app

  1. Install dependencies:
    `npm install`
  2. Start Expo:
    `npm start`
  3. Choose a target:
    - Press `a` for Android emulator/device
    - Press `i` for iOS simulator (macOS only)
    - Scan the QR code with Expo Go

  ## Migration note

  Legacy Vite/web source is still present under `src/` for reference, but it is excluded from TypeScript compilation so the native Expo app (`App.tsx`) is the active runtime entry.
